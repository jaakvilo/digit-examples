Tagasi näidete juurde

Koodimine Codexi abil

Codex on OpenAI koodiagent, mis aitab olemasolevas projektis koodi lugeda, muuta, testida ja parandada. Seda saab kasutada paarisprogrammeerijana terminalis või rakenduses, või anda talle taustal väiksemaid arendustöid.

Kiire töövoog

Ava projektikaust

Kõige parem on töötada Git-repos. Enne alustamist vaata, mis seisus projekt on: git status.

Anna Codexile konkreetne ülesanne

Kirjelda, mida muuta, kus see asub ja kuidas tulemust kontrollida. Hea ülesanne on väike: üks parandus, üks leht, üks funktsioon või üks test.

Lase Codexil koodibaasi uurida

Codex loeb faile, otsib mustreid ja pakub lahenduse. Kui ta teeb muudatusi, kontrolli neid diff’ist ja küsi vajadusel selgitust.

Käivita testid või programm

Palun Codexil käivitada sobivad testid, lintimine või käsitsi kontroll. Kui test ebaõnnestub, lase tal viga uurida ja parandada.

Vaata muudatus üle ja salvesta

Agent ei asenda vastutust. Loe muudatused läbi, testi tähtsad kasutuslood ning tee commit alles siis, kui tulemus on arusaadav.

Hea ülesande näide

Lisa sellele repo avalehele uus kaart alamprojekti jaoks. Tee eraldi kaust `minu-naide/` koos lihtsa `index.html` failiga. Hoia kujundus sama stiiliga nagu teistel kaartidel. Pärast muudatust kontrolli, et kõik kohalikud lingid töötavad.

Milleks Codex sobib?

  • Võõra koodibaasi selgitamine.
  • Väikeste funktsioonide lisamine.
  • Vigade otsimine ja parandamine.
  • Testide kirjutamine ja käivitamine.
  • README, juhiste ja kommentaaride korrastamine.

Head harjumused

  • Tee enne suuremat tööd uus Git branch.
  • Küsi Codexilt enne muutmist plaani, kui ülesanne on segane.
  • Ära anna paroole, privaatseid võtmeid ega tundlikke andmeid prompti.
  • Ära nõustu pimesi suurte kustutuste või refaktoritega.
  • Küsi lõpus: mis muutus ja kuidas seda kontrolliti?

Oluline

Codex võib koodi muuta ja käske käivitada. Õpilastega kasutades tasub alustada väikestest projektidest ning hoida Git ajalugu alles, et alati saaks vaadata, mis muutus.

Näidiskäsud õpilasele

git clone https://github.com/jaakvilo/digit-examples.git cd digit-examples git status # Seejärel ava projekt Codexis ja küsi: # "Selgita selle repo struktuuri lühidalt." # "Lisa uus lihtne näide eraldi kausta." # "Kontrolli, et HTML-lingid töötavad."
